# 🌧️ Rainfall Trend Analysis and Pattern Detection in India

## 📌 Project Overview

This project focuses on analyzing rainfall patterns across different states in India using historical data. The aim is to identify trends, seasonal variations, and deviations from normal rainfall.

The analysis helps in understanding climate behavior and can support better planning for agriculture and water resource management.

---

## 🎯 Objectives

* Analyze state-wise rainfall distribution
* Identify monthly and yearly rainfall trends
* Compare actual rainfall with normal rainfall
* Detect deviations and unusual rainfall patterns
* Generate meaningful insights using data visualization

---

## 📂 Dataset Information

* Source: India Data Portal
* Type: State-level rainfall dataset
* Features include:

  * Date
  * State Name
  * Actual Rainfall
  * Normal Rainfall
  * Deviation
  * RFS (Rainfall related metric)

---

---

## 📊 Key Analysis Performed

* Monthly rainfall trend analysis
* State-wise rainfall comparison
* Rainfall distribution analysis
* Deviation analysis
* Correlation analysis using heatmap
* Outlier detection using boxplots

---

## 💡 Key Insights

* Rainfall is highest during monsoon months (June–September)
* Some states consistently receive higher rainfall
* Several regions show below-normal rainfall trends
* Extreme values indicate irregular rainfall patterns
* Rainfall varies significantly across states and time
